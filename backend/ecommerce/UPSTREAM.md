# Upstream template source

The original 25 ecommerce scene template files in `upstream_templates/` are
imported from [liangdabiao/ecom-details-image](https://github.com/liangdabiao/ecom-details-image),
commit `1ec867b743179af3598db55388f65287c4e04de1`.

The upstream README declares the project under the MIT License. The application
does not use the upstream API script. Its template concepts are adapted for
ELLE handbags by `backend/services/ecommerce_service.py`, while all image calls
continue to use this application's configurable relay API.
